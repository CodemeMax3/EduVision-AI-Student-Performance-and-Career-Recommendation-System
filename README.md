## 📌 Project Overview

EduVision is an AI-powered web application that predicts student academic performance and recommends suitable career paths using Machine Learning. The platform integrates multiple ML models with Django REST APIs, React, MySQL, and Google Gemini AI to provide intelligent academic insights.

---

## ✨ Key Features

- 📊 Early Academic Performance Prediction
- 📈 Academic Progress Prediction
- 🏆 Student Performance Classification
- 🎯 Career Recommendation System
- 🤖 Gemini AI Career Advisor
- 📑 PDF Report Generation
- 📧 Email Notifications
- 📜 Prediction History
- 📱 Responsive Modern UI

---

## 🧠 Machine Learning Models

| Model | Type | Purpose |
|------|------|---------|
| Early Score Prediction | Regression | Predicts future academic performance |
| Academic Progress | Regression | Predicts semester progress |
| Performance Category | Classification | Classifies students into performance levels |
| Career Recommendation | Classification | Suggests suitable career paths |

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/CodemeMax3/EduVision-AI-Student-Performance-and-Career-Recommendation-System.git
```

### Backend

```bash
cd backend

python -m venv .venv

pip install -r requirements.txt

python manage.py runserver
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
